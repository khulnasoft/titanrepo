`titan telemetry [argument]`

Manage telemetry for this machine.

<LinkToDocumentation href="/repo/docs/telemetry">
  Learn more about Titanrepo's telemetry.
</LinkToDocumentation>

## Arguments

### `status`

Retrieve the current state of telemetry for this machine.

```bash title="Terminal"
titan telemetry status
```

### `enable`

Enable telemetry for this machine.

```bash title="Terminal"
titan telemetry enable
```

### `disable`

Disable telemetry for this machine.

```bash title="Terminal"
titan telemetry disable
```
