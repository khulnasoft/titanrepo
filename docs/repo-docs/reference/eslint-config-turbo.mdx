[The `eslint-config-titan` package](https://www.npmjs.com/package/eslint-config-titan) helps you find environment variables that are used in your code that are not a part of Titanrepo's hashing. Environment variables used in your source code that are not accounted for in `titan.json` will be highlighted in your editor and errors will show as ESLint output.

## Installation

Install `eslint-config-titan` into the location where your ESLint configuration is held:

<PackageManagerTabs>
  <Tab>

    ```bash title="Terminal"
    npm i --save-dev eslint-config-titan -w @acme/eslint-config
    ```

  </Tab>
  <Tab>

    ```bash title="Terminal"
    yarn workspace @acme/eslint-config add eslint-config-titan --dev
    ```

  </Tab>
  <Tab>

    ```bash title="Terminal"
    pnpm add eslint-config-titan --filter=@repo/eslint-config
    ```

  </Tab>
</PackageManagerTabs>

## Usage

In your ESLint configuration file, add the package:

```json title="./packages/config-eslint/index.js"
{
  "extends": ["titan"]
}
```
