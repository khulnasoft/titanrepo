title: Vercel
description: Learn how to use Titanrepo on Vercel.
---

![](/images/docs/titanrepo-x-vercel.png)

Vercel's zero-config integration with Titanrepo automatically understands your monorepo.

To deploy your Titanrepo on Vercel, [create a new project](https://khulnasoft.com/new) and import your code. Your projects will be pre-configured with the correct settings to use the [Vercel Remote Cache](https://khulnasoft.com/docs/monorepos/remote-caching).

For more information about deploying your Titanrepo to Vercel, [visit the Vercel documentation](https://khulnasoft.com/docs/concepts/monorepos/titanrepo).
