<Steps>
<Step>
## Install Titanrepo

Install `titan` globally so you can conveniently run `titan` commands in your terminal from anywhere in your repository.

<Tabs items={["npm", "yarn", "pnpm"]} storageKey="selected-pkg-manager">
  <Tab value="npm">
  ```bash title="Terminal"
  npm install titan --global
  ```

  </Tab>
  <Tab value="yarn">
  ```bash title="Terminal"
  yarn global add titan
  ```

  </Tab>
  <Tab value="pnpm">
  ```bash title="Terminal"
  pnpm install titan --global
  ```

  </Tab>
</Tabs>

To learn more about installing `titan`, see the [installation guide](/repo/docs/getting-started/installation).

</Step>
